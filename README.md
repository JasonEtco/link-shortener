# link-shortener

It make the link shorter.

## Running

Install dependencies:

```sh
npm install
```

Copy the sample `.env` file and replace it with your settings

```sh
cp .env.sample .env
$EDITOR .env
```

Run the migrations

```sh
npm run migrate
```

Start the application:

```sh
npm start
```

## Developing

Run tests:

```sh
npm test
```
